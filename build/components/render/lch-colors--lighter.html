Lighter Color

This demo uses color-mix(), part of the level 5 CSS color specification to lighten a black-like colors in increments of 10%.

Note: This demo only works in Firefox Nighttly and Safari TP122 or later with the `css color-mix()` experimental feature enabled as of 11/25/21. Because it is a demo, do not use in production!
