The New Defaults

A replacement for the standard CSS named color system. Provides more visually appealing hues for many CSS keywords; adds new colors with more relevant and easily remembered names. Inspired by Ingrid Sundbergs color thesaurus and the work of Adam Morse. The original From Dudley Storey, is available from Github
